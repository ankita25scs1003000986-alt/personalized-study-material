Personalized Study Material....
This project is an AI-based system that recommends personalized study material to students.
## FEATURES
- Identifies weak topics  
- Suggests suitable resources (videos, notes, questions)  
- Saves time and improves learning  

## TECHNIQUES USED
- K-Means Clustering  
- TF-IDF & Cosine Similarity  

## OUTCOMES
- Better exam preparation  
- Personalized learning experience
